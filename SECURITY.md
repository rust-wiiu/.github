# Security

If you identify security concerns within the provided software, please contact any maintainer directly. Note that security is not the primary focus of this project; niche or complex exploits may remain unaddressed if the fix imposes an unreasonable maintenance burden.

---

The Wii U is an insecure and untrusted platform. As with all sideloaded applications, do not run or install software from sources you do not trust. Malicious apps have full control over the console (OS, disk, network, etc.)
